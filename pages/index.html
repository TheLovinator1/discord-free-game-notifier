Discord Free Game Notifier


This site is used for hosting the JSON files for the discord-free-game-notifier bot.


Send webhook to Discord when a new game releases on Steam, Epic Games Store, Ubisoft Connect or GOG.

JSON files: ubisoft.json | epic.json | steam.json
